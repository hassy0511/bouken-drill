## A. 即修正バッチ（設計判断が不要なもの）

### A-1. 出題重複の修正【バグ】
- 原因：重複排除キーが「問題文＋選択肢の並び」のため、同じ漢字・同じ時刻でも
  誤答選択肢の組が違うと別問題と判定されすり抜ける
- 対応：各 `gen` が「問題の実体キー」(`q.key`)を返すようにする
  （漢字→その字、はんたい→ペア、時計→時刻、計算→式）。`buildQuestions` は `q.key` で排除

### A-2. タップ効果音
- 選択肢ボタン・テンキー・主要ボタンに軽い「ポッ」というクリック音を追加
- 「押せたかどうか分からない」問題の解消。既存の押し込みアニメと合わせて手応えを出す

### A-3. 場面別BGM＋クリアファンファーレ
| 場面 | 音 |
|---|---|
| マップ／ずかん | map曲（現状どおり） |
| クイズ中 | エリア別曲（現状どおり） |
| **クリア結果画面** | **BGM停止 → ファンファーレ（強化版）→ 勝利の短いループ** |
| 不合格の結果画面 | BGMなし。やさしい「おしい」ジングルのみ |
| エンディング | ファンファーレ＋ボス曲 |

### A-4. 時計の10分きざみ
- 出題時刻を「0・10・20・30・40・50分」に拡張
- 表記は「3じ」「3じ 30ぷん」の数値統一（**「はん」は廃止**。口語的なため）
- 短針は分に応じて中間の角度まで進める（本物の時計と同じ）
- 誤答は「長針短針の読み違え」「10分ズレ」など、ありがちな間違いから生成

### A-5. 国語系の即時増量（コード変更なし・words.json）
- はんたいことば：30 → 60ペア
- 1年漢字：g1a+g1b=64語 → **配当80字を完全収録**
- カタカナ：40 → 60語

## B. 要判断（推奨案つき）

### B-1. ステージ構成の再編（計算連続の緩和）
現状は序盤に算数5連続（もり→はま→かわ→どうくつ→たに）で単調。

- **案1：既存17ステージの並べ替えのみ**（ことば系を前に割り込ませ交互に近づける）
- **案2（推奨）：並べ替え＋「総決算とりで」2ステージ新設（計19）**
  - #6ごろ「みずうみの とりで」：それまでの範囲ミックス（好評だった総決算を増やす）
  - #14ごろ「そらの みはりだい」：中盤までの全ミックス
  - 新設分はシーン・宝物SVGを新規作成（品質基準準拠）

進捗はステージid単位で保存しているため、**並べ替えても子どもの進捗は消えない**。

### B-2. 周回＋やり込み =「チャレンジぼうけん」（推奨仕様）
- 全クリア後、マップ最上部に「⚔️ つぎの ぼうけんへ」ボタンが出現
- タップで**2周目開始**：ステージのクリア状態をリセット（マップをもう一度最初から）。
  ずかん・ベスト星・累計成績・なかまの名前は**保持**。周回数を記録
- **2周目以降はタイムアタック**：ステージごとにタイムを計測し自己ベストを記録。
  ノーミスクリアで「きんメダル」（★とは別枠のコレクション）
- 起動時は**常に「つづきから」**（選択画面を出さない。親に中断されても続きから再開）
- 船長の口上：「こんどは はやとき しょうぶだ！ まけないぜ〜！」
- 保存キー：`bk_state` に `runN` / `runCleared` / `bestTime` / `medals` を**追加**
  （既存キーの変更なし。初回移行時は現在のクリア状況を1周目として引き継ぐ）

### B-3. マップの見た目刷新（モック画像あり）
方向性：
1. ジグザグ往復 → **うねる一本道**（ノード位置を有機的に配置、間隔にも変化）
2. 点線 → **実線の道**（縁取り＋足あと点線の重ね。「道」に見せる）
3. エリア帯（四角い色帯）→ **地形のうねる帯**＋**エリアゲート看板**（「ちてい へ」）
4. ノードの間に**景観デコ**（木・波・水晶・雲・さんご等の小SVG）を散らす
5. クリア済みノードに**旗が立つ**

候補：るいぎご（にてる ことば）／かぞえかた（1ぴき・2わ・3だい）／みじかい ぶんの よみとり
