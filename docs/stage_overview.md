## マップ（v7.5で刷新）

- ジグザグ往復 → **うねる一本道**（縁取り＋地面＋足あと点線の3層描画）
- エリアは**うねる地形帯**＋**ゲート看板**（「ことば へ」等）で区切る
- ノードの間に**景観デコ**（木・波・石柱・水晶・岩・雲・さんご・あわ）をエリア別に配置
- クリア済みノードには**旗**が立つ
- 位置は `XS[]`（うねる配置%）。ステージ追加時は XS と AREAS を必ず更新
